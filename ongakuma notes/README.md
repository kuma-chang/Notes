## VENV

- Set up venv

    ``` bash
    python3 -m venv tutorial-env
    ```

- Start up venv

    ```bash
    source tutorial-evn/bin/activate
    ```

- Exit venv

    ```bash
    deactivate
    ```

- export requirements

    ```bash
    pip freeze > requirements.txt
    ```

- install requirements.txt

    ```bash
    pip install -r requirements.txt
    ```
